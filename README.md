# Dio_Desafio_Primeiro_Copilot
Documentação do desafio de criar o primeiro Copiloto no Microsoft Studio

# Objetivo
Este projeto tem por objetivo descrever o processo de criação de um agente Copilot no Microsoft Studio.

# Passos

1. Acessar o site: https://copilotstudio.microsoft.com/
2. Se logar com uma conta existente
3. Escolher a forma de criar o agente:
   3.1 Descritiva: utilizar a Inteligência Artifical para descrever o agente a ser criado.
   3.2 Baseada em modelo: Escolher um modelo pré-existente para ser usado como base para a criação do agente;
   3.3 Em brano: Iniciar a criação de um agente do Zero.
Para este projeto, vamos utilizar a opção baseada em Modelo selecionando o modelo de Benefícios.
![image](https://github.com/user-attachments/assets/7778bc2c-b7e2-434d-93cd-d87ab942991c)
4. Na tela seguinte, sao definidos o nome, a descrição instruções do agente
![image](https://github.com/user-attachments/assets/e56f0f6e-d3d5-4af9-a7a7-bd423cd21cbc)
5. Uma vez definido os textos, basta clicar em "Criar", e o agente será criado pela plataforma.
